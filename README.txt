# 🏰 Luxury Estates Maharashtra

A premium real estate platform for discovering high-quality residential and commercial properties across Maharashtra, India.

---

## 🌟 Features

- Browse properties in Mumbai, Pune, Nagpur, and Nashik  
- Filter by property type, BHK, and price range  
- Send property inquiries with budget and timeline  
- Easy contact with real estate experts  
- Admin dashboard for managing properties and leads  
- Agent dashboard with call & WhatsApp integration  
- User dashboard for tracking inquiries and wishlist  
- Real-time database using Firebase Firestore  
- Fully responsive design for all devices  

---

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript  
- Firebase Firestore  
- Font Awesome Icons  
- Google Fonts
