# 🏰 Luxury Estates Maharashtra

A premium real estate platform for commercial and residential properties across Maharashtra, India.

## 🌟 Features

- **Property Listings** - Browse commercial & residential properties across Mumbai, Pune, Nagpur, Nashik
- **Advanced Filters** - Filter by property type, BHK, price range
- **Inquiry System** - Direct property inquiries with budget and timeline
- **Contact Form** - Easy contact with real estate experts
- **Admin Dashboard** - Complete property management (Properties, Inquiries, Contacts)
- **Agent Dashboard** - Lead management with call/WhatsApp integration
- **User Dashboard** - Track inquiries and wishlist
- **Real-time Database** - Firebase Firestore integration
- **Mobile Responsive** - Fully responsive design

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript
- Firebase Firestore (Database)
- Font Awesome Icons
- Google Fonts
